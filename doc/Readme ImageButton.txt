DESCRIPTION:
------------

TWinButton is a Button that admits a bitmap. This button descends from windows buttons so it takes
new windows XP look if using


NEW PROPERTIES:
---------------

* TransparentColor
  Set transparent color

* Transparent
  Setup transparent mode

* ImageButton
  Normal image

* ImageDisabled
  Disable image

* ImageMouseOver
  Mouse over image

* ImageDown
  Image when you click

* Enabled

* AutoSize

* Clipping
  If set, button chang only when mouse are not over transparent color.
