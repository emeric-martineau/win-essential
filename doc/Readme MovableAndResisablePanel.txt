DESCRIPTION
-----------
Panel who can move and resize


NEW PROPERTIES:
---------------

* SizeBorderOfObliqueArrow : number of pixel to border who show oblique cursor.

* HeightResizable : Height's panel can be resize.

* WidthResizable : Width's panel can be resize.

* MinimalHeight : minimum size of height.
